Sincronização entre dispositivos

Cole abaixo a configuração do seu projeto Firebase. Suas tarefas vão sincronizar em tempo real entre desktop e iPhone.

Como obter a configuração?
  1. Acesse console.firebase.google.com e crie um projeto (grátis)
  2. No menu, clique em Firestore Database → Criar banco de dados → modo de teste
  3. Volte ao painel inicial, clique no ícone </> (Web) para registrar um app web
  4. Copie o objeto firebaseConfig e cole abaixo
Use o mesmo código no celular e no desktop para que os dados sincronizem.
